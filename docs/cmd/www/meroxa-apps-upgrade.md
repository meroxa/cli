## meroxa apps upgrade

Upgrade a Turbine Data Application

```
meroxa apps upgrade [--path pwd] [flags]
```

### Examples

```
meroxa apps upgrade --path ~/code
```

### Options

```
  -h, --help          help for upgrade
      --path string   path where application exists (current directory as default)
```
